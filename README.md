## Project Overview

This project predicts medical insurance costs using machine learning.  
The model analyzes factors such as age, BMI, smoking status, region, and number of children to estimate the expected insurance charge.

The application is deployed as a web application using Flask, allowing users to enter their information and receive real-time predictions.
